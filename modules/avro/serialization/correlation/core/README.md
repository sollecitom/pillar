# Avro Serialization Correlation Core

Avro serdes for the full invocation context model: `InvocationContext`, `Access`, `Actor` variants, `Authentication` variants, `AuthorizationPrincipal`, `Roles`, `Origin`, `ClientInfo`, `Session`, `AccessScope`, `Customer`, `Tenant`, `Toggles`, and `Trace`. Mirrors the JSON correlation serialization module for Avro-based messaging.
